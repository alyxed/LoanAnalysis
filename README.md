Using a synthetic loan dataset from Kaggle, I wanted to 1) find the highest impact loan approval factor and 2) ensure the dataset could be reasonably used for machine learning for predicting loan approval.

Data was processed in PostgresSQL then analyzed using Python.

The attached CSV file contains the cleaned dataset.
Loan Dataset.pdf is a presentation regarding the data and findings.
loan-analysis.ipynb is a Jupyter notebook containing all Python code.
